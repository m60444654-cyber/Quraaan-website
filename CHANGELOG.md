# دليل المساهمة

شكراً لاهتمامك بالمساهمة في تطبيق القرآن الكريم!

## كيفية المساهمة

### الإبلاغ عن الأخطاء
1. تأكد من أن الخطأ لم يتم الإبلاغ عنه مسبقاً
2. استخدم قالب الإبلاغ عن الأخطاء
3. أضيفي لقطات الشاشة إن أمكن

### اقتراح ميزات جديدة
1. تحقق من عدم وجود اقتراح مشابه
2. اشرح الميزة المقترحة بالتفصيل
3. اشرح أهمية الميزة للمستخدمين

### المساهمة بالكود
1. Fork المشروع
2. إنشاء فرع جديد
3. Commit التغييرات
4. Push إلى الفرع
5. افتح Pull Request

## معايير الكود
- اتبع أسلوب JavaScript ES6+
- استخدم التعليقات المناسبة
- اكتب كوداً نظيفاً وسهل القراءة
- اختبر الكود قبل الإرسال

## الترجمة
- نرحب بترجمة التطبيق إلى لغات أخرى
- استخدم ملفات JSON للترجمة
- تأكد من دقة الترجمة

## التوثيق
- قم بتوثيق جميع الوظائف الجديدة
- تحديث README عند الضرورة
- أضف تعليقات على الكود المعقد

## الاختبار
- اختبر على متصفحات متعددة
- اختبر على أجهزة مختلفة
- اختبر ميزات عدم الاتصال

## التواصل
- كن محترماً في جميع المناقشات
- استخدم اللغة العربية الفصحى
- ساعد الآخرين عند الحاجة

## الرخصة
جميع المساهمات مرخصة تحت رخصة MIT.
